# Password Generator

## Purpose
This project is a beginner Python program that generates passwords using standard Python modules.

## Features
 Generates memorable passwords using random words
 Generates random passwords using letters, numbers, and symbols
 Saves generated passwords with date and time

## How to Use
Run the program and choose the type of password you want.
The password will be printed and saved to a text file

## Files
password_generator.py: main program
Memorable/Generated_Passwords.txt: stores memorable passwords
Random/Generated_Passwords.txt: stores random passwords

## Libraries Used
random
 string
os
datetime
